1.2.2 Find the equations of \vec{A}D,\vec{B}E,\vec{C}F where \vec{D},\vec{E},\vec{F} are midpoints of the triangle \vec{A}BC. The vertices \vec{A},B,C are:
\begin{align}
	\vec {A}= &\myvec{1\\-1};\\
	\vec {B}= &\myvec{-4\\6};\\
	\vec {C}= &\myvec{-3\\-5};
\end{align}

Given this information and using the midpoint values found in the questions 1.2.1, the midpoints D,E,F are:

\begin{align}
	\vec {D}= &\myvec{\frac{-7}{2}\\\frac{1}{2}};\\
	\vec {E}= &\myvec{-1\\-3};\\
	\vec {F}= &\myvec{\frac{-3}{2}\\\frac{5}{2}};
\end{align}

The equation for \vec{AD} is given by
\begin{align}
\vec{x} &= \vec{A} + k\vec{m}
\end{align}

where

\begin{align}
\vec{m} &= \vec{D} -\vec{A}\\
&= \myvec{\frac{-7}{2}\\ \frac{1}{2}} -\myvec{1\\ -1}\\
&= \myvec{\frac{-9}{2}\\\frac{3}{2}}
\end{align}

Hence we get,

\begin{align}
\vec{AD}: \vec{x} = &\myvec{1\\-1} + k \myvec{\frac{-9}{2}\\\frac{3}{2}}
\end{align}

The parametric equation for \vec{BE} is given by
\begin{align}
\vec{x} &= \vec{B} + k\vec{m}
\end{align}

where

\begin{align}
\vec{m} &= \vec{E} -\vec{B}\\
&= \myvec{-1 \\ -3} -\myvec{-4\\ 6}\\
&= \myvec{3 \\-9}
\end{align}

Hence we get,
\begin{align}
\vec{BE}: \vec{x} = &\myvec{-4\\6} + k \myvec{3\\-9}
\end{align}

The parametric equation for CF is given by
\begin{align}
\vec{x} &= \vec{C} + k\vec{m}
\end{align}

where

\begin{align}
\vec{m} &= \vec{F} -\vec{C}\\
&= \myvec{\frac{-3}{2}\\\frac{5}{2}} -\myvec{-3\\ -5}\\
&= \myvec{\frac{3}{2}\\\frac{15}{2}}
\end{align}

Hence we get,
\begin{align}
\vec{CF}: \vec{x} = &\myvec{-3\\-5} + k \myvec{\frac{3}{2}\\\frac{15}{2}}
\end{align}

So the  equations of AD,BE,CF are:

\begin{align}
\vec{AD}: \vec{x} = &\myvec{1\\-1} + k \myvec{\frac{-9}{2}\\\frac{3}{2}}\\
\vec{BE}: \vec{x} = &\myvec{-4\\6} + k \myvec{3\\-9}\\
\vec{CF}: \vec{x} = &\myvec{-3\\-5} + k \myvec{\frac{3}{2}\\\frac{15}{2}}
\end{align}
